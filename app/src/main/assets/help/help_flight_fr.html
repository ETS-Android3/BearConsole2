Aide pour l'écran vol

Le vol est divisé en 2 écrans:

- L'écran graphique qui permet d'afficher toutes les courbes de vol. sur cet écran vous pouvez decider quelles courbes vous voulez afficher.

- Informations du vol. Sur cet écran vous avez un résumé du vol. Vous pouvez également exporter toutes les courbes au format CSV.
