Configuration du module 3DR

De manière à pouvoir faire de la telemetrie longue distance vous devez remplacer votre module bluetooth par un module telemetrie 3DR.

A noter que la batterie de votre appareil Android doit etre suffissement puissante pour allimenter le module 3DR. Envisager l'utilisation d'une batterie externe si vous avez un module puissant

Vous devez configurer votre module de maniere à ce qu'il puisse fonctionner avec votre altimètre.

A noter que ces modules sont en general pre-configurés à 57600 bauds. Cependant votre altimètre par defaut sera configuré à 38400 bauds vous devez donc vous assurer de vous connecter à la bonne vitesse.

A noter que les modules 3DR sont vendu par paire donc il sera necessaire de configurer à la fois l'emetteur et le recepteur

ATTENTION: assurez vous de ne pas débrancher le module lorsque vous faites les opérations qui suivent

Charger la configuration du module

Branchez le module dans votre appareil Android en utilisant un cable OTG ou un adaptateur USB qui peut fonctionner avec micro USB ou USB type C.

Ensuite cliquez sur [charger cfg] , cela effectuera des tentatives de connexion au module à differentes vitesses jusqu'à ce qu'il se connecte. Puis la configuration de module sera chargée de manière à ce que vous puissiez la changer

Sauvegarder la configuration du module

Changez les paramètres que vous voulez changer. Assurez vous que vous avec choisi la vitesse qui fonctionnera avec votre carte et cliquez [Sauver cfg]. Cela sauvegardera la configuration. En cas d'erreur lors de la sauvegarde celle-ci sera annullée

A noter que vous devrez repeter la meme operation pour chaque module
