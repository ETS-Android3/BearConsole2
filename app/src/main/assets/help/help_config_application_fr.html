Paramétrage de l'application Bear Console

La configuration est répartie en 2 écrans

Générale

- Langue de l'application(non utilisé pour le moment car la langue de l'application est la langue du téléphone)

- Couleur du graphique

- Unité de l'application

- Couleur de fond du graphique (attention à ce que cela ne soit pas la même que la couleur du graphique!!!)

- Taille de la police

- Baud rate (Ceci est pour le câble USB, cela doit être identique au baud rate de l'altimètre)

- Le type de connexion qui peut etre bluetooth ou USB si vous voulez faire de la télémétrie longue distance ou lire les données avec un câble )

- Bibliothèque graphique (permet de choisir entre 2 bibliothèques graphique; attention pour les versions 8 et suivantes d'Android votre téléphone crashera si vous n'utilisez pas la bibliotheque MPAndroid chart)

- Multiple Main/drogue Permet de définir plus d'une sortie main ou droque

- Support USB complet Activez le uniquement si vous utilisez un câble USB. Si vous utilisez un module de télémétrie longue distance ne l'activez pas pour éviter de corrompre la configuration de votre altimètre

Configuration télémétrie

Permet de sélectionner quels événements de la télémétrie sont dit. Permet de choisir la voix de la télémétrie. Si vous voulez tester la voix de la télémétrie cliquez sur le bouton [Tester la voix]

Lorsque vous avez terminé cliquer sur le bouton sauvegarder. Vous pouvez également quitter la config sans sauvegarder ou réinitialiser la config par défaut.
