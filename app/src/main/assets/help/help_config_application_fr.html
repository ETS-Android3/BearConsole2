Paramétrage de l'application Bear Console

La configuration est répartie en 2 édrans

Géneral

- Langague de l'application(non utilisé pour le moment car la langue de l'application est la langue du téléphone)

- Couleur du graphique

- Unité de l'application

- Couleur de fond du graphique (attention à ce que cela ne soit pas la meme que la couleur du graphique!!!)

- Taille de la police

- Baud rate (Ceci est pour le cable USB,cela doit etre identique au baud rate de l'altimètre)

- Connection type can be bluetooth of US if you want to do long distance telemetry)

- Graphics library (allow to choose between 2 lib type, be carefull that on Android 8 and above you will crash your phone if you do not use the MPAndroid chart lib)

- Multiple Main/drogue Alow you to define more than one output as a main or a drogue

- Full USB support Only enable it if you use USB with a cable. If using longue range telemetry do not use it to avaoid corrupting your altimeter configs

Telemetry config

Allow to configure which telemetry event are said. Allow also to change the telemetry voice. If you want to test the telemetry voice just press the [Test voice] button

Lorsque vous avez terminé cliquer sur le bouton sauvegarder. Vous pouvez également quitter la config sans sauvegarder ou réinitialiser la config par défaut.
