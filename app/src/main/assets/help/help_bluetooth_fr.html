Choix du module bluetooth

Vous serez ammené à choisir un module bluetooth si aucun n'a été selectionné ou si vous voulez vous connecter à un altimetre different.

Vous devez d'abord appairer tous les modules bluetooth de vos altimètres dans la configuration de votre appareil Android. Faites attention à bien les differencier de maniere à vos connecter au bon altimetre

A partir de la liste des modules configuré selectionnez celui utilisé par votre altimetre, quitter l'ecran et connectez vous à nouveau. Si vous avez choisi le mauvais module la connexion ne s'etabliera pas.
